# 刑法知识体系图

应试还得技术流，每一科你都值得更好的。好，接下来呢请大家把书翻到刑法的知识体系图。在讲这个图之前先讲一讲我们授课的思路。

整个刑法备考过程中有两种学习方法。第一种是从细节到细节，这种学习方法通常效率会比较低。你从细节着手到细节结束，学到最后会发现基本上一盘散沙，一团乱麻。

那与之相对应的第二种学习方法，那就是我建议各位在每个板块每章一定要先关注这一章的框架，从框架到逻辑，从逻辑到细节。所以我经常讲那么框架关联逻辑，逻辑关联细节。那这样的话就如同我们建造一座房子一样。你只要把每个房间功能区划分合理，你不同的东西放在不同的位置，等你自己去找它的时候，还是比较好找到的。如果你乱糟糟的全部堆在一起，当你碰到一个具体的知识点，想一想它在什么位置可能都很难。

所以我们先看一看刑法知识体系图，这是整个刑法的大框架。刑法无非是解决定罪和量刑的问题。哪怕是非法学的同学，你看到一些法制剧或者是法制节目里面都会涉及到构成何罪，怎么去量刑，到底是判死刑无期还是其他。那既然整个刑法是解决定罪和量刑的问题，定罪量刑会有一些指导的思想，是在什么理念下，什么思想下去定罪去量刑。由此产生了刑法的两大板块。

第一个板块是总则编，向后翻一页就是分则总则。它主要解决的问题是定罪量刑的一些方法论，怎么去定罪，如何去量刑，或者说定罪量刑要考虑哪些具体的因素。

总则呢又可以分成四个小板块。第一个板块是刑法论，我们也称之为序论。它是将刑法中的一些共性问题，一些理念性的问题放在了一起，又称序论。那么比如序论中，它就会告诉我们刑法的概念、刑法的形式、刑法的特征、刑法的体系，还包括刑法的立法目的等等。

那这个共性问题解决之后，在基本思想的指导下，我们要学会怎么去定罪，如何去量刑。而整个犯罪论它就解决两个字，定罪。它就在告诉我们定罪的方法是什么。

这个犯罪论呢又可以分成五个子板块。你可以设想一下，如果你是个法官，现在让你对被告人张三定罪量刑，那你怎么去定罪量刑呢？站在定罪角度，你首先要知道什么是犯罪。各位如果什么是犯罪，你都不知道你怎么去定罪，疯了，那不可能的呀。所以基础问题我们首先要搞明白犯罪到底是什么。

当你知道什么是犯罪以后，第二步让我去定罪，你总归要给我一个定罪的标准，或者说是一个公式。哎，满足标准，满足公式打个勾有罪，不满足打个叉无罪。所以当你知道什么是犯罪之后，第二步我们要讨论定罪标准，犯罪构成。这个犯罪构成你把它反过来看，就是构成犯罪，也就是构成犯罪的条件。

那么以定罪标准犯罪构成为中心。我们知道了什么是犯罪了也知道定罪的标准是什么了。现在就可以对被告人去定罪了，我们看看他到底有罪还是有还是无罪。

但是呢，一个人在犯罪的过程中，他有可能是将犯罪进行到底。比如张三将李四杀死了，犯罪推进到底完成了。但也有可能在犯罪的过程中，因为种种原因停止下来，砍了两刀被害人跑了，砍了两刀自己被警察抓了，并没有以一己之力将犯罪进行到底，这个时候怎么办？他就涉及到故意犯罪的形态问题，所以以定罪标准为中心，如果构成犯罪，以他为中心，要讨论犯罪在时间上的推进，你到底推进了哪一步，离犯罪的完成还有多远，这就是犯罪的形态问题。

但是定罪标准，犯罪构成，它往往解决的是一个人构成犯罪怎么办？那请问如果张三入户盗窃偷东西，李四望风，你对这个望风的李四怎么去定罪呢？此时就必须要引入共同犯罪理论。大家可以设想一下，如果没有共犯理论，你对李四怎么定罪？人家李四就在门口站着，什么都没干。我们国家可没有规定什么望风罪，没有规定什么站门罪，没有类似的规定。

那此时引入共犯理论，那就是以定罪标准为中心，空间上的分布就是共犯。那共犯为什么是空间分布呢？我们在共同犯罪这个板块会讲的，现在呢主要讲一个大致的框架。因为一旦构成共同犯罪，原则上要按照它的作用大小区分主犯、从犯、胁从犯。各位如果五个人一块实施了抢劫，那么一旦构成共同犯罪，我们认为作用大的是在金字塔的顶端，我们称之为主犯。然后紧接着小一点的，我们称之为从犯，最小的我们可能会称之为胁从犯。所以构成共犯之后要区分作用大小，就如同一个空间分布一样。

因此我们解决第一个问题，什么是犯罪？第二个问题，定罪标准。犯罪构成以它为中心，时间推进是形态，以它为中心，空间分布是共犯。那最后你作为一个法官，让你去定罪人家到底构成一个罪还是两个罪，如果构成数个罪，要不要数罪并罚？所以我们还要去解决罪数问题。

由此整个犯罪论其实就解决五大问题。一什么是犯罪？二定罪标准，犯罪构成，三时间推进犯罪形态，四空间分布共同犯罪，五定罪几个罪罪数问题。这个逻辑应该是很清晰的。

那构成犯罪之后，原则上要承担刑事责任啊，由此就会进入刑事责任这个阶段。承担什么样的刑事责任，如何去承担？当这个板块比较抽象，那么我们在刑事责任这个板块再具体去展开。

刑事责任主要的实现方式是刑罚。我们看电视剧都知道啊，张三犯抢劫罪，处五年有期徒刑并处罚金。那这个处五年有期徒刑，他其实就是一种刑罚，叫刑事处罚。那因此整个刑罚论要解决的问题，那就是量刑问题。比如量何种刑，怎么去量刑，哪些影响轻重，这就是整个刑罚论讨论的。

那我们会发现整个总则最核心的问题是在犯罪论刑法能不能学好。在总则角度关键就看犯罪论，它也是最难的最重要的。那这种学习方法的好处就在于你在大脑中其实已经建构了一个框架。你知道总则分成四个板块，刑法论、序论、犯罪论、刑事责任论、刑罚论。四个板块分清楚之后，然后每个板块下面有自己的逻辑，这是个大的框架，每个板块都有逻辑，用逻辑再去整合每个板块的细节，这样的话细节问题我们的记忆才会更长久，处理才会更妥当。好，这个小点我们先暂时介绍到这儿。

那总则结束过之后，我们就会学会怎么去定罪，如何去量刑。但是定罪定何罪，那你当然要考虑刑法分则规定的有哪些罪名。比如我们国家没有规定溺婴罪。大家可能也看过类似新闻吧，小孩子刚出生，然后呢父母将孩子扔掉淹死了，但是我们国家没有规定溺婴罪。但是刑法第232条规定了故意杀人罪。这个时候您就只能按照故意杀人罪定罪处罚，不能按照溺婴罪定罪处罚。

那由此我们刑法分则，它其实是对总则的具体的运用，定罪定何罪，你要看分则规定的哪些罪名。

那如果我们对犯罪进行分类，可以对分则大致分成这三个板块，包括侵害个人法益的犯罪。当然法益是第一次出现啊，我解释一下这两个字，法益指的是法律保护的利益啊，你可以简单称之为合法利益。那么侵犯个人法益犯罪主要是集中在两章，包括侵犯公民人身权利、民主权利罪，侵犯财产罪，这是刑法分则的第四章和第五章的规定。

那么侵犯社会法益的犯罪有三章，具体是刑法分则第二章危害公共安全罪，第三章破坏社会主义市场经济秩序罪，还有第六章妨害社会管理秩序罪。

侵害国家法益的犯罪有五章，具体到刑法分则，第一章是危害国家安全罪，第七章是危害国防利益罪，第八章贪污贿赂罪，第九章渎职罪，第十章军人违反职责罪。这是刑法分则十章大致的一个分类。

那么鉴于我们目前考试的要求，在国家法益这一块，大家可以在危害国防利益罪旁边画个叉，军人违反职责罪旁边画个叉，这两章不在考试范围之内。由此刑法分则有十章罪，纳入大纲考察的有八章，去年新大纲又增加了15个罪名。所以我们目前刑法分则压力的确是比去年要更大一些。

好，到了这一步，大家应该是对整个总则和分则的框架有了基本的认识。有些细节问题我们后面再去讨论。好，下面我们接着向后翻，请大家将书翻到第五页，我们开始看刑法论。
